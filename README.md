# TiendaPro - E-commerce Frontend (Módulo 2)

Este proyecto es el desarrollo de la primera versión (MVP) de una tienda en línea enfocada en productos de computación y gaming, creada para el **Módulo 2** del Bootcamp de Academia Sustantiva.

## 🚀 Características
- **Diseño Responsivo (Mobile-First):** Optimizado para dispositivos móviles (≤ 420px) y escritorio (≥ 1024px) utilizando el sistema de grillas de **Bootstrap 5**.
- **Navegación Semántica:** Estructura completa con HTML5 semántico (`header`, `nav`, `main`, `section`, `article`, `footer`).
- **Carrito de Compras (Simulado):** 
  - Contador en el Navbar actualizado en tiempo real mediante manipulación del DOM con **JavaScript**.
  - Persistencia de datos utilizando **LocalStorage** para mantener el conteo al recargar la página.
  - Notificaciones dinámicas ("Toasts") al agregar productos.
- **Flujo de Usuario:** Navegación fluida entre la página de Inicio y la sección de Computación.

## 🛠️ Tecnologías utilizadas
- **HTML5** (Semántica y estructura)
- **CSS3** (Variables personalizadas, Flexbox y Grid)
- **Bootstrap 5** (Componentes Navbar, Cards, Badges y CDN)
- **JavaScript Vanilla** (Eventos, DOM, Funciones y LocalStorage)
- **Font Awesome** (Iconografía profesional)
